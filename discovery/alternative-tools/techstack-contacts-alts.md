# C1c: Free Alternatives to BuiltWith/Wappalyzer + ZoomInfo/Lusha

> Research date: 2026-07-23
> Context: Direct research (agent launcher unavailable)

## PART 1: TECH STACK DETECTION

### 1.1 unbuilt.app
- **URL:** https://unbuilt.app / https://github.com/yavorsky/unbuilt.app
- **Type:** Open source (100% free, no limits)
- **Free tier limits:** Completely free. Analyzes bundled/minified code in real-time. Detects bundlers, UI libraries, frameworks, analytics, monitoring, platforms. CLI tool available for local analysis (no data sent to servers).
- **Commercial use:** Allowed (open source)
- **Sufficiency for 25 niches:** SUFFICIENT -- unlimited use, CLI + web, no API key needed
- **Deprecation risk:** Low-Medium -- active development, growing community

### 1.2 Open Tech Explorer
- **URL:** https://github.com/turazashvili/openexplorer.tech
- **Type:** Open source + Chrome extension
- **Free tier limits:** Free. Tech stack detection, metadata analytics, smart search, RESTful API. Chrome extension for real-time analysis.
- **Commercial use:** Likely allowed
- **Sufficiency for 25 niches:** SUFFICIENT -- has API for programmatic use
- **Deprecation risk:** Medium

### 1.3 Populr Stacks Chrome Extension
- **URL:** Chrome Web Store
- **Type:** Free browser extension (open source)
- **Free tier limits:** Unlimited per-site analysis. Privacy-first -- no data sent to remote servers. Shows code snippets as evidence per detection.
- **Commercial use:** Allowed
- **Deprecation risk:** Low

### 1.4 Whopper (npm CLI)
- **URL:** https://socket.dev/npm/package/whopper
- **Type:** Open source CLI tool
- **Free tier limits:** `npm install -g whopper`. Unlimited local tech detection. Wappalyzer-like functionality from command line.
- **Commercial use:** Allowed (MIT)
- **Deprecation risk:** Medium

### 1.5 Wappalyzer Free Tier (We have this)
- **URL:** https://www.wappalyzer.com
- **Free tier limits:** 50 credits/month. 1 credit per technology lookup. Credits expire after 60 days. 5 website alerts, sample lead lists, browser extension.
- **Commercial use:** Allowed
- **Deprecation risk:** Low

## PART 2: B2B CONTACT DATA (ZoomInfo/Lusha)

### 2.1 Snov.io Free Tier
- **Free tier limits:** 50 credits/month for email finding/verification. 100 email recipients for campaigns. **No API access on free tier.** 1 mailbox warm-up slot.
- **Commercial use:** Allowed
- **Sufficiency for 25 niches:** INSUFFICIENT -- no API on free tier limits programmatic use
- **Deprecation risk:** Low

### 2.2 6sense Chrome Extension
- **URL:** Chrome Web Store
- **Free tier limits:** 50 contact credits. Works on LinkedIn profiles, websites. Finds emails, direct dials, firmographics from 15+ sources.
- **Commercial use:** Allowed
- **Sufficiency for 25 niches:** INSUFFICIENT -- 50 contacts only
- **Deprecation risk:** Medium

### 2.3 ZeroBounce Email Verification
- **URL:** https://www.zerobounce.net
- **Free tier limits:** 100 free credits/month (recurring). 99.6% accuracy. SOC 2 Type 2 and ISO 27001 certified.
- **Commercial use:** Allowed
- **Sufficiency for 25 niches:** BORDERLINE -- 100/month is enough for periodic list cleaning
- **Deprecation risk:** Low

### 2.4 NeverBounce
- **Free tier limits:** No recurring free tier. One-time demo only (~1,000 credits as one-off test).
- **Commercial use:** Allowed on paid plans
- **Sufficiency for 25 niches:** INSUFFICIENT -- no recurring free tier
- **Deprecation risk:** Low
